- [ ] **Standard:** X01, Cricket

- [ ] **Practice:** Around the Clock

### Shell

- [ ] **Four tabs** — Play, Players, Activity, Settings — **no Modes tab**

- [ ] Player detail: **no** Export, **no** Training Partner (**custom bots OK**)
