### Description (full listing — replace live 1.0 body)

```
Dart Buddy is a free darts scorekeeper for iPhone — fast scoring, clean Apple-native UI, and no ads. Your matches and players stay on your device.

SEVEN GAME MODES
Start from Play → Change mode:

CLASSICS
• X01 (301/501) with checkout suggestions
• Cricket — Normal and Cut Throat

PARTY MODES
• Baseball — inning runs and bull playoff tie-breakers (2+ players)
• Killer — pick your number, eliminate opponents (3+ players)
• Shanghai — score each round's target segment (2+ players)

CO-OP
• Raid — close shield segments, then finish the boss together (1–3 human players)

PRACTICE
• Around the Clock — solo drill from 1 through bull

MATCH TOOLS
• Undo, resume in-progress games, and forfeit with standings
• Match history and per-mode statistics
• Preset and custom skill bots (where the mode allows)

BUILT FOR REAL PLAY
• VoiceOver and Dynamic Type support
• Local-first — no account required; works offline after install
• Optional haptics and sound feedback

Free. No ads. No in-app purchases.
```
