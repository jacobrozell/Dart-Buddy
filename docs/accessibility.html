Dart Buddy is a darts scorekeeper for X01 and Cricket. We design for clarity during play: high-contrast scores, large tap targets on the scoring pad, and support for common iPhone accessibility settings.

This page describes how Dart Buddy works with iOS accessibility features. Availability may vary by screen; we continue to improve coverage with each release.

Supported features

Dark Interface

Dart Buddy supports light, dark, and system appearance.

  • In the app: Settings → Appearance → choose Light, Dark, or System.
  • Or use iOS Settings → Display & Brightness → Dark with Appearance set to System.

Reduced Motion

When Reduce Motion is on, celebratory animations on the match summary screen and certain feedback banners are simplified or disabled.

  • iOS Settings → Accessibility → Motion → Reduce Motion

VoiceOver

Core flows include spoken labels for scoring controls, player score cards, match setup, history, players, and settings. Gameplay announcements cover events such as busts, leg wins, and cricket target closures.

  • iOS Settings → Accessibility → VoiceOver
  • On the scoring pad, keys are announced with full dart names (for example, “Triple 20” rather than “T20”).

Larger Text (Dynamic Type)

Text scales with your system text size. Layouts adapt at larger sizes so setup, scores, and primary actions remain reachable.

  • iOS Settings → Accessibility → Display & Text Size → Larger Text

Contrast and non-color cues

Scoreboard typography and controls are designed for legibility in light and dark mode. Turn state, cricket target closure, and status feedback use text and icons—not color alone.

Gameplay tips

  • Scoring pad: Buttons meet a 52×52 pt target in gameplay for easier tapping.
  • Undo: Available from the match screen toolbar after submitting a turn.
  • Turn total caller: Optional spoken visit total after submit — Settings → Turn total caller.
  • Orientation: Portrait and landscape are supported on iPhone.

App Store accessibility features

Dart Buddy declares support on the App Store for VoiceOver, Dark Interface, Sufficient Contrast, Larger Text, Differentiate Without Color Alone, and Reduced Motion on iPhone and iPad. Each feature is intended to work across core tasks: onboarding, settings, match setup, X01 and Cricket scoring, match summary, players, and activity (history and statistics).

Known limitations

  • At the largest accessibility text sizes, some secondary stats may require scrolling on smaller phones.
  • Dart Buddy does not include video content; Captions and Audio Descriptions do not apply.
  • If any core task does not work with an assistive feature on your device, please report it using the contact below.
