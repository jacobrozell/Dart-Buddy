Häufige Fragen

Wie lösche ich alle meine Daten?
Öffnen Sie Einstellungen → Alle lokalen Daten zurücksetzen → bestätigen. Dadurch werden Spieler, Spielverlauf und Einstellungen auf diesem Gerät dauerhaft entfernt.

Brauche ich ein Konto?
Nein. Dart Buddy funktioniert offline; Daten bleiben auf Ihrem Gerät.

Gibt es Werbung?
Nein. Dart Buddy ist kostenlos und werbefrei.

Wo werden meine Daten gespeichert?
Auf Ihrem iPhone oder iPad. Optionale Diagnose in App-Store-Versionen beschreibt unsere Datenschutzerklärung.
