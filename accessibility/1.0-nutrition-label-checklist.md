# 1.0 Nutrition Label checklist (~30 min)

Timed manual script for **App Store Connect Accessibility Nutrition Labels** and RC sign-off. Run on a **physical iPhone and iPad** (both are declared in Connect), **Release build**, **no launch args**.

**Declared labels (1.0):** VoiceOver · Dark Interface · Sufficient Contrast · Larger Text · Differentiate Without Color Alone · Reduced Motion — every row in the scorecard must pass on **both** devices before submit.

**Related:** [`Manual_todo.md`](Manual_todo.md) (full per-screen detail) · [`../docs/release/release_checklist.md`](../docs/release/release_checklist.md) §6 · [`../roadmap/release/QA-Signoff-RC1.md`](../roadmap/release/QA-Signoff-RC1.md) · hosted page [`../docs/accessibility.html`](../docs/accessibility.html)

## Part 2 — VoiceOver (~15 min)

**Enable:** Settings → Accessibility → **VoiceOver** → On  
**Gestures:** swipe right = next; double-tap = activate; two-finger Z = dismiss

### Flow A — X01 end-to-end (~6 min)

| Done | Step | Listen for |
|------|------|------------|
| [ ] | Play → setup (Alice + Bob) → **Start Match** | Mode chip; START; roster “selected” |
| [ ] | Score one visit (e.g. 20, 20, 20) → **Submit** | Full dart names; visit on score card |
| [ ] | **Undo Last Turn** | Labeled; state reverts |
| [ ] | Arm **TRIPLE** → tap 20 | “Triple 20” on pad |
| [ ] | (Optional) force a **bust** | Bust announcement (not silence) |
| [ ] | Reach **Match summary** | Winner + stats; **New match** CTA named |
| [ ] | Flow A complete without sighted help | [ ] Pass · [ ] Fail |

### Flow B — Cricket (~5 min)

| Done | Step | Listen for |
|------|------|------------|
| [ ] | New match, Cricket (Normal), Alice + Bob | Setup labels |
| [ ] | Submit several marks | “Triple 20”, “Miss”, “Double bull” |
| [ ] | Close one target (3 marks) | Closure / board update announcement |
| [ ] | **Cut Throat:** change chip in setup → one turn | Pad operable; no unlabeled controls |
| [ ] | Flow B complete without sighted help | [ ] Pass · [ ] Fail |

### Flow C — Tabs + Settings (~4 min)

| Done | Step | Listen for |
|------|------|------------|
| [ ] | **Activity → History** | Filters; row combined summary |
| [ ] | **Activity → Statistics** | Mode filter; table rows |
| [ ] | **Players** → Alice | Name, stats, Edit |
| [ ] | **Settings** scroll | Theme, sound, haptics labeled |
| [ ] | **Reset All Data** → **Cancel** only | Destructive button + alert buttons named |
| [ ] | Flow C complete without sighted help | [ ] Pass · [ ] Fail |

## If something fails

| Failure | Likely area | Blocks label |
|---------|-------------|--------------|
| START hidden at AXXXL | `SetupHomeView` layout | Larger Text |
| Score clipped in X01 | `PlayerScoreCard` | Larger Text |
| VO dead end on pad | `DartNumberPad` labels | VoiceOver |
| Cricket closure silent | `CricketBoardView` announcements | VoiceOver, non-color |
| Trophy still animates | `MatchSummaryScreen` reduceMotion | Reduced Motion |

File defect in QA sign-off § Defects; ship without that label.
